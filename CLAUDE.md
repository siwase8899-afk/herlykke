# HERLYKKE 웹앱 (04-web-mvp)

4050 여성 수면·뇌건강 커뮤니티 마켓. Next.js 16 + React 19 + Tailwind 4 + Supabase. 배포: herlykke.vercel.app (GitHub siwase8899-afk/herlykke).

## Design System
시각/UI 작업 전 **반드시 `DESIGN.md`를 먼저 읽는다.** 폰트·컬러·여백·미감·데이터 시각화 톤이 거기 정의돼 있다.
핵심: "차분한 신뢰" — 따뜻한 크림 베이스(#F7F3EC), 영문·숫자 Fraunces 세리프 + 한글 Pretendard, 세이지/라벤더 + 테라코타 온기, 제네릭 이모지·평면 저대비 카드 금지, 데이터는 초승달/방사형 링. DESIGN.md에서 벗어날 땐 사용자 승인 필요.

## 출시 설정
실연동(Supabase 글 저장 등)은 `LAUNCH-SETUP.md`의 Vercel 환경변수 설정 필요.
